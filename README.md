# Sarcasm-Dataset

Detect Sarcasm in news headlines. Refer to the [.ipynb](./Sarcasm%20Detection.ipynb) file for the entire code. Dataset linked [here](https://storage.googleapis.com/laurencemoroney-blog.appspot.com/sarcasm.json).

## Framework Used:

Model trained using `TensorFlow v2.2.0`
